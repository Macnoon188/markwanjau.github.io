Week 1 – Cybersecurity Homelab Series

Inside My Proxmox Homelab: Architecture, Purpose, and Practical Use Cases

A real-world cybersecurity lab powering threat detection, learning, and hands-on engineering

Published: Week 1, Monday • Author: Jonathan Kirui • Tags: Homelab • Proxmox • Cybersecurity • Virtualization

Over the past year, I've been building a Proxmox-based homelab designed not just for experimentation, but as a functional foundation for cybersecurity, virtualization, and scalable personal projects. In this post, I’ll break down my homelab’s architecture, the problems it solves, and how I’ve used it to grow hands-on experience that aligns with real-world enterprise needs.

🛠️ My Homelab Stack at a Glance

• Hypervisor: Proxmox VE (Debian-based)
• Compute: Refurb Dell rackmount servers (64GB RAM, 2x Xeon)
• Storage: ZFS pools using HDD/SSD hybrid
• Networking: pfSense firewall with VLAN segmentation
• Containers / VMs:
– Wazuh SIEM node
– Kali Linux for pen-testing
– TurnKey file server (Debian)
– Ubuntu + Docker environment for dev/testing
– Windows VM for domain testing

“A homelab isn’t just a playground — it’s a proving ground for real-world cybersecurity readiness.”

🎯 Why I Chose Proxmox

Proxmox gives me a clean web interface, built-in clustering, automated backups, low resource overhead with LXC containers, and native support for ZFS and GPU passthrough. Most importantly — it’s free, open-source, stable, and trusted across enterprise environments.

🔒 My Security Layers

• Wazuh 4.5 for endpoint monitoring & CVE detection
• pfSense gateway with segmentation & OpenVPN
• Syslog forwarding from all nodes
• Scheduled vulnerability scans

Practical Wins

This stack helped me detect and mitigate threats like CVE-2025-29824 in real time using community rules, layered defense, and proactive monitoring.

📚 Practical Learning Outcomes

This lab setup helped me:
• Build hands-on cybersecurity engineering skills
• Simulate enterprise-grade incident response
• Practice CVE mitigation methodologies
• Prepare for certifications like CYSA+ and CISA

👨‍💻 Final Thoughts

A homelab doesn’t need to be expensive or perfect — it needs purpose. Mine has become a live training environment, a cybersecurity research platform, and a place where I sharpen skills that translate directly to the real world.
